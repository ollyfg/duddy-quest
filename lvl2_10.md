# Level 2 Task 10 — Enchanted broom shopkeeper enemies

## Context

Several Diagon Alley shops have enchanted brooms and mops that try to
sweep Dudley out the front door. These are light combat encounters that
add variety between the pixie swarms and the Draco boss fight.

## Requirements

### Enchanted broom enemy type

Create a new enemy variant using the hostile NPC system:

| Property | Value |
|---|---|
| `is_hostile` | `true` |
| `movement_mode` | `PATROL` with 2 patrol points (sweeping back and forth) |
| `move_speed` | `90.0` (moderate) |
| `hp` | `2` |
| `contact_damage` | `1` |
| Color | Brown (`Color(0.6, 0.4, 0.2)`) |
| `knockback_force` | High — brooms push Dudley back significantly |

**Sweeping behaviour**: Brooms patrol a horizontal or vertical line in
front of a shop entrance. On contact, they deal damage AND apply extra
knockback toward the shop exit, simulating being "swept out". The
knockback direction should always point toward the nearest room exit.

### Implementation approach

- Use existing `npc.gd` with `PATROL` movement mode and 2 patrol points
- Add a new export `knockback_direction_override: Vector2` to `npc.gd`
  that, when set, replaces the normal knockback direction (which is
  calculated from collision angle) with a fixed direction. This lets
  brooms always push toward the exit.
- Alternatively, this could be a simple `npc.gd` with high
  `knockback_force` and patrol points aligned to sweep across the room.

### Placement

- 1–2 brooms in `l2_madam_malkins.tscn` (sweeping near entrance)
- 1 broom in `l2_menagerie.tscn` (sweeping near animal cages)
- 0 brooms in Gringotts and Ollivander's (those have their own
  encounters)

### Visual
- Same rectangular NPC sprite but coloured brown
- Consider adding a slight rotation to the sprite to suggest a broom
  angle (set `$Sprite.rotation` in `_ready`)

## Acceptance criteria

- [ ] Brooms patrol back and forth in shop rooms
- [ ] Contact deals damage and pushes player toward exit
- [ ] Brooms are defeatable with 2 melee hits
- [ ] Knockback direction is consistently toward the exit
- [ ] Brooms don't leave their patrol path to chase the player
- [ ] Rooms with brooms pass `check_alignment.py`
